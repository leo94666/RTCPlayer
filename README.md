# RTCPlayer 播放器

一个基于Android客户端的的RTC播放器


## 项目特点

- [ ]支持[ZLMediakit](https://github.com/ZLMediaKit/ZLMediaKit)流媒体
- [ ]支持[Janus](https://github.com/meetecho/janus-gateway)
- [X]支持[SRS](https://github.com/ossrs/srs)
